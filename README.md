# Fin Research Agent

**个人美股投资研究助手** - 多源异构数据融合 + Bull/Bear 结构化分析

## 核心特性

### 1. 多源异构数据融合
整合 5 类金融信息源，覆盖投研全场景：

| 数据源 | 内容类型 | 价值 |
|--------|---------|------|
| 久谦中台 | 专家访谈、产业链调研 | 一手信息、产业洞察 |
| AlphaEngine | 研报、业绩说明会 | 官方数据、分析师观点 |
| Twitter | KOL动态、市场情绪 | 实时舆情、事件驱动 |
| Substack | 深度Newsletter | 独立研究、另类视角 |
| AceCampTech | 共享调研纪要 | 行业分析、公司研究 |

### 2. Bull/Bear 结构化分析
不给模糊答案，而是提供多空对照的结构化分析：

```
用户问：Dell 未来业绩怎么看？

🐂 看多论点                    🐻 看空论点
├─ AI服务器强劲增长              ├─ 传统PC业务持续承压
│  来源: AlphaEngine            │  来源: AceCampTech
├─ 与NVIDIA深度合作              ├─ AI服务器毛利率低
│  来源: 久谦中台                │  来源: 久谦中台
└─ 企业AI市场领先地位            └─ 竞争加剧(HPE/联想)
   来源: Twitter                    来源: 久谦中台

⚖️ 综合判断: 中性偏多
```

### 3. 智能意图路由
根据问题类型自动选择最优数据源组合：

- 业绩会查询 → AlphaEngine 优先
- 专家访谈 → 久谦中台优先
- 行业趋势 → 久谦中台 + AceCampTech
- 社交舆情 → Twitter + Substack

### 4. 完整溯源体系
每个观点可追溯到原始来源，包含：
- 来源平台
- 文档标题
- 发布时间
- 预览链接

```

    ↓

└─────────────────────────────────┘
    ↓
┌─────────────────────────────────┐
│     多源检索层                   │  ← Step1宽检索 + Step2精准检索
│  - 公司名变体扩展               │
│  - 时效性约束                   │
└─────────────────────────────────┘
    ↓
┌─────────────────────────────────┐

    ↓

```

## 快速开始

### 1. 克隆仓库
```bash
git clone https://github.com/YOUR_USERNAME/fin-research-agent.git
cd fin-research-agent
```

### 2. 安装依赖
```bash
pip install -r requirements.txt
```

### 3. 配置环境变量（可选）
```bash
cp .env.example .env
# 编辑 .env 文件，填入你的 API Key
```

> **注意**: 不配置环境变量时，系统自动使用 Demo 模式（内置 mock 数据）

### 4. 运行 Demo
```bash
streamlit run demo/app.py
```

访问 http://localhost:8501 体验 Demo

## Demo 模式说明

本仓库提供完整的代码架构和 Demo 数据。Demo 模式下支持以下预置问题：

1. **Dell未来业绩怎么看** - 展示 Bull/Bear 多空分析
2. **HBM扩产比例** - 展示多源数据融合
3. **光模块行业趋势** - 展示智能意图路由

生产部署需要：
- 配置 Claude API Key（用于 Bull/Bear 分析）
- 配置知识库 API（用于真实数据检索）

## 创新点

1. **多源异构融合**: 首次将专家访谈、研报、社交媒体、Newsletter 等异构数据源整合到统一检索框架
2. **Bull/Bear 结构化输出**: 不给模糊答案，提供可对比的多空观点，辅助投资决策
3. **智能意图路由**: 根据问题类型自动选择最优数据源组合，提升检索精度
4. **完整溯源体系**: 每个观点可追溯到原始来源，支持深度验证
